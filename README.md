# gym-meal-plan
自分の筋トレライフに合わせて作った個人用の食事管理プランです。

筋トレに合わせた食事管理プラン。平日・土日・オフ日の3パターンで栄養を管理。

## 概要

- タンパク質・カロリー・食費を曜日タイプ別に管理
- 平日 / 土日 / オフ日の3パターン対応
- ブラウザで開くだけで使える単一HTMLファイル
- iPhoneのホーム画面に追加してPWAとして使用可能

## 使い方

1. [こちらから開く](https://ykobayashi-jp.github.io/gym-meal-plan/)
2. タブで平日・土日・オフ日を切り替え

## 技術仕様

- 単一HTMLファイル
- CSS / JavaScript のみ、サーバー不要
